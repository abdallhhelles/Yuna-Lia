# NSFW Drafts

This folder is reserved for optional NSFW runtime content packs.

Keep it separate from the main `themes/` library so the default project can stay cleaner and easier to moderate.

Suggested workflow:

1. Draft and review NSFW content here first.
2. Decide later whether the runtime should load this folder by default or only behind a guild config switch.
3. Keep clear labels for intensity, tone, and consent context.
